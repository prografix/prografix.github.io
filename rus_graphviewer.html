GraphViewer, начиная с версии 3.3, позволяет загружать файлы в формате STL. Для этого надо выбрать в меню пункт Import, затем STL. В результате появится панель открытия файлов. В ней есть возможность сделать загружаемый объект гладким ( Smooth ) или гранёным ( Flat ). Этот объект затем можно сохранить как gra- или cpp-файл. Небольшая коллекция STL-файлов находится здесь.

GraphViewer, начиная с версии 3.4, позволяет загружать файлы в формате Wavefront OBJ. Для этого надо выбрать в меню пункт Import, затем OBJ. В результате появится панель открытия файлов. Этот объект затем можно сохранить как gra- или cpp-файл. Небольшая коллекция OBJ-файлов находится здесь.
