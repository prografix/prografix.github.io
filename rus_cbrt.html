Арифметические корни

Следующие функции вычисляют действительные корни соответсвующей степени: 

double root3 ( double x );
double root4 ( double x );
double root5 ( double x );
double root6 ( double x );
double root7 ( double x );
double root8 ( double x );
double root9 ( double x );
double root10( double x );
Исходники находятся в файле mathem.cpp.

Следующая программа вычисляет корни третьей степени из комплексного числа:

void root3 ( Complex x, Complex & r1, Complex & r2, Complex & r3 );
Описание типа Complex смотрите в разделе Класс Complex.
Исходники находятся в файле complex.cpp.
