Приложение DEMO ( версия 4 )


В эту версию добавлен ввод-вывод файлов в формате Wavefront OBJ. Этот формат позволяет граням иметь произвольное к-во вершин в отличии от STL-формата. Имеется коллекция OBJ-файлов, которые можно посмотреть. Также в этой версии переключение вида изображения ( wire, flat, smooth ) перенесено из панели "Scene parameters" в главное меню ( пункт Model ).
