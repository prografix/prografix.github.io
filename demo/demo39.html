Приложение DEMO ( версия 39 )


В этой версии добавлено построение минимальной сферы, пересекающей все данные прямые. Для просмотра надо в меню выбрать пункт Demo / Minimization 3D. В новой панели надо указать: внешний объект ( сфера ) и внутренний объект ( прямые ). К-во прямых можно менять от 4 до 9. Кнопкой "Следующий" выдаётся новый набор случайных прямых.
