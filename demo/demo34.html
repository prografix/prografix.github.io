Приложение DEMO ( версия 34 )


В этой версии добавлены примеры получения некоторых пересечений на плоскости.
Для просмотра надо в меню выбрать пункт Demo / Intersection 2D. В новой панели можно указать первый ( прямая, отрезок, окружность, эллипс, многоугольник ) и второй ( прямая, отрезок, вып. многоугольник ) объекты. Для многоугольников можно выбрать к-во вершин. Кнопками "Следующий" выдаётся следующий случайный объект. Выпуклый многоугольник может иметь меньше вершин, чем было задано, т.к. в этом случае строится выпуклая оболочка случайных точек.
