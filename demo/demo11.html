Приложение DEMO ( версия 11 )


В этой версии показаны примеры вписания максимальной окружности в выпуклый и невыпуклый многоугольник. Для просмотра надо в меню выбрать пункт Demo / Max circle. В новой панели можно указать к-во вершин многоугольника ( от 3 до 12 ), в меню Function выбрать нужную функцию. Кнопка Next даёт следующий случайный многоугольник с заданным к-вом вершин. В случае выбора функции для вписания в выпуклый многоугольник полученный многоугольник будет выпуклым и к-во его вершин может быть меньше заявленного числа, т.к. в этом случае строится выпуклая оболочка. Кнопка Run не меняет многоугольник и служит для вписания окружности разными функциями.
