Приложение DEMO ( версия 47 )


Иногда в одном STL-файле бывает записано несколько многогранников, и возникает задача их разделить. В этой версии добавлен пример функции, которая это делает. Для запуска этой функции надо в меню выбрать пункт Demo / Other. В новой панеле нажмите на кнопку "OK". Затем укажите нужный файл. Разделённые объекты будут записаны в файлы ( бинарный STL ), в название которых будет добавлен суффикс "part" и номер файла.
Также в этой версии добавлена демонстрация нахождения минимальной трапеции вокруг данных точек.
Для этого надо в меню выбрать пункт Demo / Minimization 2D.
