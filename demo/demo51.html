Приложение DEMO ( версия 51 )


В этой версии показаны несколько вариантов триангуляции многоугольников на плоскости.
Для просмотра надо в меню выбрать пункт Demo / Triangulation 2D. В новой панели можно указать к-во вершин многоугольника ( от 4 до 13 ), вариант триангуляции ( none - не делать триангуляцию, SweepLine - заметающая прямая, Seidel - метод Зейделя ) и включить построение триангуляции Делоне. Кнопка Next polygon даёт следующий случайный многоугольник с заданным к-вом вершин. Кнопка Triangulate заполняет текущий многоугольник диагоналями. Кнопка Convex parts разбивает текущий многоугольник на выпуклые многоугольники.
