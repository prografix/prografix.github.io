Приложение DEMO ( версия 23 )

Для вызова панели "Аппроксимация в пространстве" надо в меню выбрать пункт Demo / Approximation 3D. В новой панели можно выбрать вариант аппроксимации: по многограннику построить шар, эллипсоид или параллелепипед. Для многогранника можно указать к-во вершин ( от 4 до 11 ). Кнопка "Следующий" даёт новый набор точек и результат аппроксимации. Пункт меню Model позволяет выбрать разные представления примеров ( в данном случае варианты Flat и Smooth выглядят одинаково ).
