Приложение DEMO ( версия 7 )


В этой версии показаны два варианта аппроксимации набора точек прямой линией на плоскости. Для просмотра надо в меню выбрать пункт Demo / Line appoximation. В новой панели можно указать к-во точек ( от 3 до 11 ). Кнопка Next points даёт следующий набор случайных точек. Точки обозначены крестиками. Красная линия получена функцией getLine2, жёлтая - функцией getLine1а.
