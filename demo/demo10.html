Приложение DEMO ( версия 10 )


В этой версии показаны несколько вариантов триангуляции многоугольников на плоскости. Для просмотра надо в меню выбрать пункт Demo / Triangulation. В новой панели можно указать к-во вершин многоугольника ( от 4 до 13 ), вариант триангуляции ( none - не делать триангуляцию, MaxMinArea - максимизировать минимальный треугольник, MaxMinAdivR2 - максимизировать минимальное качество AdivR2, MaxSumAdivR2 - максимизировать сумму качеств AdivR2, MaxMinTan - максимизировать минимальный тангенс угла, MaxSumAngle - максимизировать сумму минимальных углов ) и выбрать тип оптимизации ( глобальная - локальная ). Кнопка Next polygon даёт следующий случайный многоугольник с заданным к-вом вершин. Кнопка Triangulate заполняет текущий многоугольник диагоналями.
