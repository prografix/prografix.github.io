Приложение DEMO ( версия 3 )


В этой версии реализован ввод-вывод STL-файлов. Этот тип файлов описан в разделе Формат STL. Там же есть небольшая коллекция таких файлов, которые можно посмотреть. При просмотре можно переключать вид изображения в панели "Scene parameters" ( wire, flat, smooth ). При сохранении в файл можно выбрать бинарный или текстовый вариант.
