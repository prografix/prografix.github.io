Приложение DEMO ( версия 18 )


В этой версии добавлено вписание максимального фиксированного эллипса в выпуклый многоугольник. Для просмотра надо в меню выбрать пункт Demo / Maximization 2D. В новой панели можно указать: внешний объект ( выпуклый или невыпуклый многоугольник ) и внутренний объект ( окружность, эллипс или многоугольник ). Для многоугольников можно указать к-во вершин. У эллипса удлинение равно двум. Кнопками "Следующий" выдаётся следующий случайный многоугольник с заданным к-вом вершин. Если был выбран выпуклый многоугольник, то его к-во вершин может быть меньше заданного числа, т.к. в этом случае строится выпуклая оболочка. Для невыпуклого внешнего многоугольника вписание возможно только для окружности.
