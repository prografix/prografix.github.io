Приложение DEMO ( версия 33 )


В этой версии добавлено построение минимальной окружности охватывающей данные окружности и минимальной сферы охватывающей данные сферы.
Для просмотра минимальной окружности надо в меню выбрать пункт Demo / Minimization 2D. В новой панели можно указать: внешний объект ( окружность, эллипс фикс., эллипс своб., многоугольник, треугольник, прямоугольник или параллелограмм ), внутренние объекты ( точки или окружности ) и их к-во. Кнопками "Следующий" выдаются новые внешние или внутренние объекты. Для внутренних окружностей реализовано только нахождение минимальной окружности.
Для просмотра минимальной сферы надо в меню выбрать пункт Demo / Minimization 3D. В новой панели можно указать: внешний объект ( сфера, эллипсоид или параллелепипед ), внутренние объекты ( точки или сферы ) и их к-во. Кнопкой "Следующий" выдаётся новый набор внутренних объектов. Для внутренних сфер реализовано только нахождение минимальной сферы. Вид изображения можно менять в пукте меню Model ( виды Flat и Smooth в данном случае будут выглядить одинаково ).
