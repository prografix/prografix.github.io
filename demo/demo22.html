Приложение DEMO ( версия 22 )


В этой версии добавлены вписания максимальных прямоугольников, ромбов, параллелограммов, треугольников и четырёхугольников в выпуклый многоугольник. Для просмотра надо в меню выбрать пункт Demo / Maximization 2D. В новой панели можно указать: внешний объект ( выпуклый или невыпуклый многоугольник ) и внутренний объект ( окружность, эллипс фикс., эллипс своб., треугольник, прямоугольник, ромб, параллелограмм, четырёхугольник или многоугольник ). Для многоугольников можно указать к-во вершин. Для свободного эллипса и ромба, кроме максимума по площади ( малиновый цвет ) выводится максимум по периметру ( жёлтый цвет ). Кнопками "Следующий" выдаётся следующий случайный многоугольник с заданным к-вом вершин. Если был выбран выпуклый многоугольник, то его к-во вершин может быть меньше заданного числа, т.к. в этом случае строится выпуклая оболочка. Для невыпуклого внешнего многоугольника вписание возможно только для окружности.
