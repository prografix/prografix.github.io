Приложение DEMO ( версия 26 )


В этой версии добавлены примеры аппроксимации многоугольника кругом и прямоугольником.
Для этого надо в меню выбрать пункт Demo / Approximation 2D. В новой панели можно выбрать варианты аппроксимации: 1) по точкам построить прямую, 2) по точкам построить окружность, 3) по многоугольнику построить круг, 3) по многоугольнику построить эллипс, 3) по многоугольнику построить прямоугольник. А также можно указать к-во точек ( от 4 до 11 ). Кнопка "Следующий" даёт новый набор точек и результат аппроксимации. При построении по точкам прямой жёлтым цветом рисуется вариант getLine1a, а красным цветом вариант getLine2. При построении по точкам окружности жёлтым цветом рисуется вариант getCircle1a, а красным цветом вариант getCircle2.
